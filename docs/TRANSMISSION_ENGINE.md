# Transmission Engine

CALL-AWAKEN transmissions remain resilient:

- Bundled content renders without Supabase.
- Supabase campaigns are additive and draft-seeded by default.
- The approved `awaken-system/ads.js` file remains available as the visual/presentation definition source.
- Runtime actions are dispatched through `awaken:transmission-*` events.

Normal browsing should show at most one active popup, respect dismissal state, and use real destinations only.
